HTML is the standard language used to define what you want to display on your webpage. It stands for HyperText Markup Language.

With HTML, you can tell a browser to display some text as a heading or to draw a table for you. It’s a simple but powerful language.

In this part, we will give you all the basics that you need to know to write a simple blogpost. In the next part, we’ll look at how to integrate links and images on your page.

If you want to learn more, have a look at the HTML tutorial created by w3schools. w3schools also has a very cool editor that allows you to play around with html and immediately see results.

Blog Post Template

Here’s a simple blog post template that we’ll use to explore the anatomy of a webpage.

HTML consists of nested elements. Each element gets opened with <element> and closed with </element>.

Remember: <h1> opens a header-1 (the largest kind of heading) and </h1> closes it.

<head> and <body>

The contents of <head> are not displayed. Instead, it’s where javascript and css files are loaded and where the title of the webpage is set. That title appears in the browser tab.

The <body> is what you actually see on the webpage.

Divs

The <body> usually consists of a number of <div> (“division”) elements.

The first one renders the navigation bar. The second div contains the content of the blogpost and uses the class "column" for layout and styling.

<div class="column">
    ....
</div>
